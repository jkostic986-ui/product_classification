# product_classification

# Automatska klasifikacija proizvoda

Cilj projekta je razviti ML model koji automatski predlaže kategorije za proizvode na osnovu naziva.


## Struktura repozitorijuma

product_classification/
├─ data/                   
├─ notebooks/              
├─ scripts/                
│   ├─ train_model.py
│   └─ predict_category.py
├─ product_category_model.pkl
└─ README.md

## Kako koristiti

1. **Notebook:** `notebooks/product_classification.ipynb`  
2. **Treniranje modela:** `python scripts/train_model.py`  
3. **Interaktivna predikcija:** `python scripts/predict_category.py`
